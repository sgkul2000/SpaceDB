# TeloHive Spaces

Space discovery and booking dashboard built with Next.js 15, React 19, TypeScript strict, and Tailwind v4.

## Setup

```bash
# Install dependencies
npm install

# Start mock API (terminal 1)
npm run api

# Start dev server (terminal 2)
npm run dev
```

App runs at `http://localhost:3000`. API runs at `http://localhost:3001`.
